# excel2latexTable_MAC
A shell script to convert an Excel table to latex format using the Clipboard.

Run with mac os X (El capitan)

How to use this script:
1.Copy your table in Excel  
2.Run the script  
3.Paste into latex  


Example:  

\toprule  
Sector a & Sector b & Sector c  
\midrule  
127,0 & 26,0 & -1,1  
13,3 & 161,6 & 14,5  
11,0 & 22,1 & 18,6  
5,2 & 11,0 & 0,9  
\bottomrule\
